# TG Content Engine - Development PRD

## Project Overview
Build the TG Content Engine web application - a Telegram content automation platform for IDIGOV Real Estate (Dubai). The system automates scraping Dubai/UAE news, generating AI-written bilingual posts (EN/RU), and publishing to Telegram on a fixed schedule.

## Tech Stack
- Frontend: Next.js 15, TypeScript, Tailwind CSS 4, shadcn/ui, TanStack Query, Zustand
- Backend: FastAPI, SQLAlchemy 2.0 (async), Celery + Redis, Pydantic v2
- Database: PostgreSQL 16, Redis 7
- External APIs: Claude (stubbed), Gemini (stubbed), Telegram Bot (stubbed)
